# AI-Agents

Enterprise-grade Python project for intelligent agent development.

## Project Structure

- `src/` — Main source code
- `tests/` — Unit and integration tests
- `docs/` — Documentation

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run tests:
   ```bash
   pytest
   ```
